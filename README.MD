# 📦 Supply Chain Inventory Analysis

> Automated inventory & stock analysis dashboard built with Python & Jupyter.

Transforms raw inventory data from Excel and operational sources into
interactive KPI dashboards and automated HTML reports.

---

## 🔧 Tech Stack

| Tool | Purpose |
|---|---|
| Python · Pandas | Data loading & cleaning |
| Plotly | Interactive visualizations |
| Jupyter | Analysis & reporting environment |
| nbconvert | Export to HTML report |

---

## 📂 Project Structure

data/         → Raw Excel / CSV input files
notebooks/    → Jupyter analysis notebooks
outputs/      → Generated HTML reports

---

## 📊 Key KPIs Analyzed

- Stock level by product / category
- Inventory turnover rate
- Dead stock identification
- Reorder point alerts
- Stock variance over time

---

## 🚀 How to Use

1. Place your Excel data file in `data/`
2. Open the notebook in `notebooks/`
3. Run all cells
4. Find the HTML report in `outputs/`
